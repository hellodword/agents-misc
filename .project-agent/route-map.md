# Project Route Map

Project routes apply only to this upstream repository. They are not part of the distributed Agent Rules Kit.

## Shared kit maintenance

When touching `AGENTS.md`, `.agents/**`, `schemas/agent-rules/**`, `scripts/check-agent-rules.py`, its tests, or shared-kit documentation in `README.md`, load:

- `.project-agent/rules/agent-rules-kit.md`
- `.agents/rules/core/repo-hygiene.md`
- `.agents/rules/core/testing.md`
- `.agents/rules/toolchain/nix.md` when Just, Nix checks, or Flake validation changes

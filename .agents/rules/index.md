## Exclusions that prevent false routes

- A database mention does not select SQLite; require project evidence or an explicit choice.
- A screenshot or generic web page does not select browser E2E or AI visual review.
- CI work does not select GitHub Actions unless that platform is established and the user explicitly requests the workflow change.
- Mentioning or refusing a local Nix installer does not select the Nix workflow skill; use that skill only for a project-owned workflow described above.
- Vue applies only when the repository already uses it or the user explicitly selects it.
- A complex task does not by itself select the standalone-plan skill.
- An ordinary test, review, or status request does not authorize a commit.

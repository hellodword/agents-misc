## Severity scale

- P0: Blocks the user task or makes the UI unusable.
- P1: Serious visual, accessibility, or responsive issue.
- P2: Noticeable quality issue that should be fixed.
- P3: Minor polish suggestion.

## Categories

Use exactly these category values:

- `layout`
- `hierarchy`
- `spacing`
- `alignment`
- `typography`
- `color-contrast`
- `responsive-behavior`
- `i18n-copy`
- `accessibility`
- `state-handling`
- `consistency`
- `visual-polish`
- `possible-bug`

## Review instructions

- Cite screenshot ids.
- Provide visual evidence.
- Avoid subjective preference without evidence.
- Prefer issues that reproduce across multiple viewports/locales.
- Mark conflicts or ambiguous trade-offs as `needs-human-decision`.
